# Market Interface

# 1. Exchange Information

HTTP Request

    # Request
    GET /b2b/nomics/rank/info

response description

Name Type Status Description
code code Mandatory Status code Note:200 means success and other means failure
msg String Mandatory Prompt information
name string Mandatory The name of the exchange
description string Mandatory An exchange description
location string Mandatory The primary country the exchange operates from
logo decimal Mandatory logo
website decimal Mandatory A URL to your exchange
twitter decimal Recommended twitter
version decimal Mandatory Platform Specification version
markets decimal Mandatory true is implements ,false is not .
trades decimal Mandatory true is implements ,false is not .
ordersSnapshot decimal Mandatory true is implements ,false is not .
candles decimal Mandatory true is implements ,false is not . 
    # Response
  {
    "msg": "success",
    "code": 200,
    "data": {
        "capability": {
            "ordersSnapshot": true,
            "markets": true,
            "ticker": true,
            "trades": true,
            "candles": true
        },
        "description": null,
        "logo": "11111",
        "name": "bityard",
        "twitter": "bityard@twitter.com",
        "version": "v1.1.0",
        "website": "https://www.bydfi.com"
    }
}

# 2. Current Order Book Snapshot

HTTP Request

                        # Request
    GET /b2b/nomics/rank/orders/snapshot

request Paramters

name Type Status description
market string Mandatory market ID from the /markets

Response description

name Type Status description
code code Mandatory Status code Note:200 means success and other means failure
msg String Mandatory Prompt information
timestamp Integer Mandatory the timestamp this snapshot was created in RFC3339 in UTC
bids decimal Mandatory An array containing 2 elements. The offer price and quantity fyor each bid order
asks decimal Mandatory An array containing 2 elements. The ask price and quantity for each ask order
avgResTimeMs Long Mandatory avg response time.ms
invoke boolean Mandatory true invoke history 'false' is not invoke

	"msg": "success",
	"code": 200,
	"data": {
		"asks": [4033, 22],
		"bids": [40222, 11],
		"timestamp": "2022-03-22T15:29:32.465Z"

	},
	"avgResTimeMs": "33",
    	"invoke": false
}

# 3. Candles

HTTP Request

                    
    # Request
    GET /b2b/nomics/rank/candles

request description

name Type Status description
market string Mandatory market ID from the /markets
interval string Mandatory The period of each candle example:1m,1h, 1d

response description

name Type Status description
code code Mandatory Status code Note:200 means success and other means failure
msg String Mandatory Prompt information
open string Mandatory The opening price
close string Mandatory The closing price
low string Mandatory The low price
high string Mandatory The high price
volume string Mandatory Accumulated trading value, in quote currency
timestamp string Mandatory timestamp of the start of the candle in RFC3339 aligned to candle size in UTC
avgResTimeMs Long Mandatory avg response time.ms
invoke boolean Mandatory true invoke history 'false' is not invoke

# 6. Ticker

HTTP Request

                    
    # Request
    GET /b2b/nomics/rank/ticker

request description

name Type Status description
market string Mandatory market ID from the /markets

response description

name Type Status description
code code Mandatory Status code Note:200 means success and other means failure
msg String Mandatory Prompt information
close string Mandatory trade id
timestamp string Mandatory Timestamp of the trade in RFC3339 in UTC
volume string Mandatory volume of the asset in the base currency
high string Mandatory highest price of the asset in the quote currency as a string parseable to a positive number
low string Mandatory lowest price of the asset in the quote currency as a string parseable to a positive number
ask string Mandatory open price of the asset in the quote currency as a string parseable to a positive number
bid string Mandatory open price of the asset in the quote currency as a string parseable to a positive number
avgResTimeMs Long Mandatory avg response time.ms
invoke boolean Mandatory true invoke history 'false' is not invoke
